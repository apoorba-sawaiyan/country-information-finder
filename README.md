# Country Information Finder

A Python project that uses the REST Countries API to fetch and display country information.

## Features

- Search country by name
- Display capital city
- Display population
- Display currency
- Display country flag
- Compare two countries

## Technologies Used

- Python
- Requests
- REST API
- JSON
